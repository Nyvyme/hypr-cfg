# Nyvyme's Hyprland Config

## Basic stuff
Wallpapers are from
[pattern-wallpaper](https://github.com/Nyvyme/pattern-wallpaper).
Look into `hyprpaper.conf` file and install needed wallpapers.

GTK theme &mdash; Mint-X

Cursor theme &mdash; macOS

## Default utilities
* File manager &mdash; [nemo](https://github.com/linuxmint/nemo)
* Terminal &mdash; [ghostty](https://ghostty.org)
* Application launcher &mdash; [wofi](https://github.com/SimplyCEO/wofi)
* Browser &mdash; [zen-browser](https://zen-browser.app)
* Screenshot utility &mdash; [hyprshot](https://github.com/Gustash/hyprshot)
* Bar &mdash; [waybar](https://github.com/Alexays/Waybar)

## Keybinds
* `SUPER + Enter` &mdash; open terminal
* `SUPER + C` &mdash; kill active window
* `SUPER + M` &mdash; kill Hyprland
* `SUPER + E` &mdash; open file manager
* `SUPER + F` &mdash; toggle floating
* `SUPER + P` &mdash; open application launcher
* `SUPER + Shift + F` &mdash; open browser
* `SUPER + Shift + J` &mdash; toggle split
* `SUPER + Left arrow` / `SUPER + H` &mdash; move focus to left
* `SUPER + Right arrow` / `SUPER + L` &mdash; move focus to right
* `SUPER + Up arrow` / `SUPER + J` &mdash; move focus to up
* `SUPER + Down arrow` / `SUPER + K` &mdash; move focus to down
* `SUPER + 1` &mdash; open first workspace
* `SUPER + 2` &mdash; open second workspace
* `SUPER + 3` &mdash; open third workspace
* `SUPER + 4` &mdash; open forth workspace
* `SUPER + 5` &mdash; open fifth workspace
* `SUPER + 6` &mdash; open sixth workspace
* `SUPER + 7` &mdash; open seventh workspace
* `SUPER + 8` &mdash; open eighth workspace
* `SUPER + 9` &mdash; open ninth workspace
* `SUPER + 0` &mdash; open tenth workspace
* `SUPER + Shift + 1` &mdash; move active window to first workspace
* `SUPER + Shift + 2` &mdash; move active window to second workspace
* `SUPER + Shift + 3` &mdash; move active window to third workspace
* `SUPER + Shift + 4` &mdash; move active window to forth workspace
* `SUPER + Shift + 5` &mdash; move active window to fifth workspace
* `SUPER + Shift + 6` &mdash; move active window to sixth workspace
* `SUPER + Shift + 7` &mdash; move active window to seventh workspace
* `SUPER + Shift + 8` &mdash; move active window to eighth workspace
* `SUPER + Shift + 9` &mdash; move active window to ninth workspace
* `SUPER + Shift + 0` &mdash; move active window to tenth workspace
* `SUPER + A` &mdash; toggle special workspace (magic overlay)
* `SUPER + Shift + S` &mdash; move active window to magic workspace
* `SUPER + Scroll wheel` &mdash; scroll through existing workspaces
* `SUPER + Right button` &mdash; move window
* `SUPER + Left button` &mdash; resize window
* `SUPER + Shift + B` &mdash; execute script to launch waybar
  (you probably don't need this one)
* `PrintScr` &mdash; screenshot whole screen
* `Alt + PrintScr` &mdash; screenshot screen area
